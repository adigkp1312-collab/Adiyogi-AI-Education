# Existing AI Education Platforms — Research

## 1. Coursera

**Structure**: Three tiers — Professional Certificates (multi-course, job-ready), Specializations (deep-dive series), and individual Courses.

**Course Design**:
- Curriculum mapped to job market skills, validated by employers
- 20+ hands-on activities per certificate program
- Video lectures + quizzes + peer-graded assignments + capstone projects
- Partners: Google, IBM, Meta, DeepLearning.AI

**AI Agent Courses (2026)**:
- Google AI Professional Certificate
- AI Agent Developer Specialization
- Growing focus on agentic AI, AI-assisted design, critical thinking

**Key Takeaway**: Employer-validated curriculum structure. Portfolio-building through hands-on projects. Tiered learning paths (beginner → professional).

Source: https://www.coursera.org/courses?query=artificial%20intelligence

## 2. DeepLearning.AI (Andrew Ng)

**Structure**: Three formats:
- **Short Courses** (97 offerings) — 1-3 hours, skill-specific
- **Specializations** (14 offerings) — multi-course, 5 weeks each at 5 hrs/week
- **Courses** (10 offerings) — standard length

**Teaching Philosophy**: Visual concepts → practical coding → underlying mathematics. No prior math or rigorous coding background required.

**AI Agent Topics**:
- Multi-agent system design (CrewAI partnership)
- Agent tool execution and integration
- Voice agents and real-time interaction
- Data agents with planning capabilities
- Agent governance and safety
- API discovery and knowledge graphs

**Key Takeaway**: "Dual-track" approach — foundational specializations for depth, short courses for targeted skills. Partnership model with framework creators (CrewAI, Google, Anthropic, Databricks).

Source: https://www.deeplearning.ai/courses/

## 3. fast.ai (Jeremy Howard)

**Structure**: Single comprehensive course — "Practical Deep Learning for Coders"
- ~20 hours of video lessons
- 7 weeks at 10 hrs/week recommended pace
- Part 1 (practical) + Part 2 (foundations)

**Teaching Philosophy**: **Top-down approach** — start with complete working solutions to real problems, gradually work down to foundations. Opposite of traditional academic bottom-up teaching.

**Course Progression**:
1. Computer vision (image classification)
2. NLP (text classification, language models)
3. Tabular data analysis
4. Collaborative filtering
5. Deep dive into foundations

**Hands-on**: Interactive Jupyter notebooks on Kaggle. Companion book: "Practical Deep Learning for Coders."

**Key Takeaway**: Top-down, project-first pedagogy. Minimal prerequisites (1 year coding). Free and open-source. Strong community.

Source: https://course.fast.ai/

## 4. Udacity

**Structure**: Nanodegree programs — typically 3-4 months, project-based
- AI Programming with Python
- Artificial Intelligence Nanodegree
- **Agentic AI Nanodegree** (new 2025-2026)
- Generative AI Nanodegree

**Teaching Philosophy**: Project-based learning. Every module culminates in a portfolio-ready project. Mentorship and code review included.

**AI Agent Topics (Agentic AI Nanodegree)**:
- LangChain-powered agent development
- API integration and memory management
- Contextual response systems
- Real-world agent deployment

**Key Takeaway**: Portfolio-focused. Structured projects with mentor review. Paid but comprehensive. Industry-aligned curriculum.

Source: https://www.udacity.com/course/agentic-ai--nd900

## 5. Hugging Face AI Agents Course (Free)

**Structure**: 4 main units + 3 bonus units
- 1 week per chapter, 3-4 hours/week
- Theory → Hands-on → Use cases → Final challenge

| Unit | Topic | Description |
|------|-------|-------------|
| 0 | Onboarding | Setup tools and platforms |
| 1 | Agent Fundamentals | Tools, Thoughts, Actions, Observations, LLMs, chat templates |
| 2 | Frameworks | smolagents, LangGraph, LlamaIndex implementation |
| 3 | Use Cases | Real-life agent building |
| 4 | Final Assignment | Build agent for benchmark, compete on leaderboard |
| Bonus 1 | Fine-tuning LLMs | Function-calling specialization |
| Bonus 2 | Observability | Agent evaluation and monitoring |
| Bonus 3 | Agents in Games | Pokemon battle agents |

**Hands-on**: Hugging Face Spaces with pre-configured environments. Student leaderboard for competitive evaluation.

**Key Takeaway**: Community-driven, open-source, free with certification. Progressive complexity. Competitive leaderboard adds gamification. Best free AI agents course available.

Source: https://huggingface.co/learn/agents-course/en/unit0/introduction
